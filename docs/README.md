# ZETIN 드론 프로젝트

이 저장소는 드론 제어, 데이터 분석 및 펌웨어 개발에 중점을 둔 ZETIN 드론 프로젝트를 위한 코드 및 문서들을 포함합니다.

## 디렉토리 구조

*   `docs/`: 본 README, 커밋 메시지, 프레젠테이션을 포함한 프로젝트 문서.
*   `logs/`: CSV 형식의 비행 및 드론 운용 로그 저장.
*   `scripts/`: 드론 제어, 분석 및 튜닝을 위한 Python 스크립트.
*   `firmware/`: 드론 펌웨어 개발 관련 파일들을 포함합니다.
    *   `firmware/src/`: 메인 드론 펌웨어 소스 코드.
    *   `firmware/lib/`: 펌웨어에서 사용되는 사용자 정의 라이브러리.
    *   `firmware/include/`: 펌웨어용 헤더 파일.
    *   `firmware/examples/`: 실제 드론에 업로드될 중요한 `.ino` 파일들을 포함하는 다양한 테스트 및 예제 펌웨어 프로젝트 (예: DSHOT, PWM, IMU 테스트).
*   `test/`: 현재 `tcp_test.py`를 포함한 일반 테스트 스크립트.

## 시작하기

### 펌웨어 개발

드론 펌웨어는 `firmware/examples/` 디렉토리에 있는 `.ino` 파일들을 기반으로 개발됩니다. 이 파일들은 드론에 업로드되는 실질적인 펌웨어 코드입니다.

1.  `firmware/examples/` 디렉토리에서 원하는 `.ino` 파일을 선택합니다.
2.  해당 `.ino` 파일을 아두이노 IDE 또는 다른 적절한 개발 환경에서 열고 드론에 업로드합니다.

### 드론 제어 및 분석

`scripts/` 디렉토리에는 드론 데이터와 상호 작용하고 분석하기 위한 Python 스크립트가 포함되어 있습니다.

1.  `scripts/` 디렉토리로 이동합니다.
2.  원하는 Python 스크립트를 실행합니다 (예: 제어를 위한 `python Drone_Control_Dualsense.py`, 분석을 위한 `python Drone_Analasys.py`).

### 로그

비행 및 드론 로그는 `logs/` 디렉토리에 자동으로 저장됩니다. 이 로그는 비행 후 분석 및 튜닝에 활용될 수 있습니다.
