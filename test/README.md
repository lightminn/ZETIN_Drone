# Test (테스트)

이 디렉토리는 시스템의 개별 모듈이나 네트워크 통신 기능을 독립적으로 검증하기 위한 테스트 스크립트들을 포함합니다.

## 포함된 테스트

- **`tcp_test.py`**: 드론과 PC 간의 TCP 통신이 정상적으로 이루어지는지 확인하기 위한 스크립트입니다. 패킷 송수신 지연(Latency) 및 무결성을 검사합니다.

## 실행 방법

1. 테스트하려는 서버(드론 또는 PC)를 대기 상태로 둡니다.
2. 스크립트를 실행하여 결과를 확인합니다:
   ```bash
   python tcp_test.py
   ```

## 목적
새로운 통신 프로토콜을 구현하거나 대역폭을 변경했을 때, 전체 시스템에 적용하기 전 이 디렉토리의 스크립트를 통해 안정성을 먼저 확인하는 것을 권장합니다.
